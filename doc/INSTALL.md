# search-file-content

Search for text inside files across directories, with path filters and live content preview.


## Install from PYPI

The homepage in pipy is https://pypi.org/project/search_file_content/

```bash
pip install --upgrade search_file_content
```

or (recommended for desktop applications)

```bash
pipx install search_file_content
```


Using:

```bash
search-file-content
```

## Install from source
Installing `search-file-content` program

```bash
git clone https://github.com/trucomanx-desktop/SearchFileContent.git
cd SearchFileContent
pip install -r requirements.txt
cd src
python -m build
pip install dist/search_file_content-*.tar.gz
```
Using:

```bash
search-file-content
```

## Uninstall

```bash
pip uninstall search_file_content
```
